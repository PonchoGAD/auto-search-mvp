# Auto Semantic Search MVP — API Specification

## 1. Назначение
API предоставляет семантический поиск по авто-объявлениям.
Пользователь вводит запрос обычным языком (RU),
система:
- извлекает параметры,
- ищет по смыслу,
- возвращает сгруппированный результат:
  **лучшие варианты + объяснение + источники**.

MVP работает в read-only режиме, без пользователей и авторизации.

---
